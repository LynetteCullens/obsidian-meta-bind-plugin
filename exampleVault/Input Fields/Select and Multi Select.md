### Select
```meta-bind
INPUT[select(
option(1, option 1), 
option(2, option 2), 
option(3, option 3), 
showcase
):select]
```

```meta-bind
INPUT[select(
option(1, option 1), 
option(false, option 2), 
option(null, option 3), 
showcase
):select2]
```
